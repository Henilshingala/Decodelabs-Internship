Generated by running `python classification.py` against `dataset.csv`
(150 samples, 50 per species, 80/20 stratified train-test split, `k=5`,
`random_state=42`).

## Analysis

- **`setosa`** is classified with perfect precision and recall — its
  features are linearly separable from the other two species in this
  dataset.
- **`virginica`** has perfect precision (every sample predicted as
  `virginica` really was `virginica`) but recall of only 0.80 — 2 of 10
  true `virginica` samples were misclassified as `versicolor`.
- **`versicolor`** has lower precision (0.83) because it "absorbed" those 2
  misclassified `virginica` samples, but perfect recall (every true
  `versicolor` sample was correctly identified).

## How to Reproduce

```bash
cd Project-2-Data-Classification
pip install -r requirements.txt
python classification.py
```

The script prints this report to the console and regenerates
`confusion_matrix.png` on every run. Because `random_state=42` is fixed, the
split — and therefore these exact numbers — are reproducible.
